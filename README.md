# ExhibitIQ 🎨

A modern, comprehensive gallery management system built with Next.js, React, and Tailwind CSS. ExhibitIQ provides powerful tools for managing artwork inventory, auctions, analytics, and digital floor experiences.

## ✨ Features

### 🖼️ Artwork Management
- **Inventory**: Complete catalog management with support for Artwork, Collectibles, Memorabilia, and Objects
- **Artist Profiles**: Comprehensive artist database and profile management
- **Digital Floor**: Interactive digital displays with QR code integration
- **Featured Artworks**: Highlight and showcase special pieces

### 🏛️ Gallery Operations
- **Auction Management**: End-to-end auction planning and execution
- **Patron Management**: Customer relationship management and engagement
- **Record Vault**: Secure storage for gallery documentation and records
- **Gallery Settings**: Customizable gallery configuration and preferences

### 📊 Analytics & Insights
- **Gallery Analytics**: Comprehensive performance metrics and reporting
- **Revenue Tracking**: Financial analytics and revenue optimization
- **User Activity**: Visitor engagement and behavior analysis
- **Top Performers**: Identify best-selling artists and artworks

### 🎯 Digital Experience
- **QR Code Integration**: Active QR codes for artwork information
- **Digital Floor Performance**: Real-time display analytics
- **Interactive Elements**: Modern, responsive user interface
- **Mobile-First Design**: Optimized for all device types

## 🚀 Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React & Heroicons
- **Charts**: Recharts
- **Package Manager**: pnpm

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/blacckoscar/ExhibitIQ.git
   cd ExhibitIQ
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Run the development server**
   ```bash
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
